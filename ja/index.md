# ヘルプセンター

{% include lang-switcher.html current="ja" slug="index" %}
</div>

IFG GEO Optimizerは、ShopifyマーチャントがChatGPT、Perplexity、Gemini、Claude、Google AI OverviewsなどのAI検索エンジンにとって、商品カタログをより読みやすく、引用しやすく、推薦されやすくするお手伝いをします。これは**GEO**(Generative Engine Optimization、生成エンジン最適化)であり、従来のSEOとは異なる分野です。構造化データ、機械可読なコンテンツ、そしてクローラーのアクセス性が、キーワードランキングよりも重要になります。

このサイトには、アプリを利用している方、導入を検討している方、Shopifyアプリストア向けにレビューしている方、いずれにとっても必要な情報がすべて揃っています。

## アプリの機能

- 不足しているschema.org構造化データを**ワンクリックで修正**できる**カタログ監査**、加えて`llms.txt`と`agents.md`の生成、Organizationスキーマ。
- **クローラーシミュレーター** — AIボット(GPTBot、ClaudeBot、PerplexityBot)が商品ページで実際に何を見ているかを、JavaScriptなしで、ボットごとの`robots.txt`チェックも含めて確認できます。
- **コンテンツチャンキングスコア** — マークアップの有無だけでなく、商品コピーが生成エンジンにとってどれだけ事実を抽出しやすいかを評価します。
- **AI支援によるFAQ生成**と**コンテンツリライト**ツール — どちらも公開前に必ずお客様自身がレビュー・承認します。
- **Link Coach** — カタログ全体にわたる内部リンク分析。
- **AI可視性トラッキング** — Claude、ChatGPT、Gemini、Perplexityがあなたのストアに言及しているかを毎週チェックし、あわせてAI経由の流入トラフィックを計測します。
- **競合比較** — 同じGEOシグナルを、競合の商品ページと並べて比較できます。

すべての機能の詳しい説明は[はじめに](getting-started.html)を、正確なプラン制限については[FAQ](faq.html)をご覧ください。
