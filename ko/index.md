# 헬프 센터

{% include lang-switcher.html current="ko" slug="index" %}
</div>

IFG GEO Optimizer는 Shopify 판매자가 ChatGPT, Perplexity, Gemini, Claude, Google AI Overviews와 같은 AI 검색 엔진이 상품 카탈로그를 더 쉽게 읽고, 인용하고, 추천할 수 있도록 도와줍니다. 이는 **GEO**(Generative Engine Optimization, 생성형 엔진 최적화)로, 전통적인 SEO와는 다른 분야입니다. 키워드 순위보다 구조화 데이터, 기계가 읽을 수 있는 콘텐츠, 크롤러 접근성이 더 중요합니다.

이 사이트에는 앱을 사용하든, 평가하든, Shopify 앱스토어 심사를 위해 검토하든 필요한 모든 정보가 담겨 있습니다.

## 앱이 하는 일

- 누락된 schema.org 구조화 데이터를 **원클릭으로 수정**하는 **카탈로그 감사**, 그리고 `llms.txt`와 `agents.md` 생성 및 Organization 스키마.
- **크롤러 시뮬레이터** — AI 봇(GPTBot, ClaudeBot, PerplexityBot)이 상품 페이지에서 실제로 무엇을 보는지 JavaScript 없이 확인하며, 봇별 `robots.txt` 검사도 포함됩니다.
- **콘텐츠 청킹 점수** — 마크업 존재 여부뿐 아니라, 생성형 엔진이 상품 설명에서 사실을 얼마나 쉽게 추출할 수 있는지를 평가합니다.
- **AI 지원 FAQ 생성**과 **콘텐츠 재작성** 도구 — 두 기능 모두 게시 전 항상 사용자가 직접 검토하고 승인합니다.
- **Link Coach** — 카탈로그 전반의 내부 링크 분석.
- **Visibility AI 추적** — Claude, ChatGPT, Gemini, Perplexity가 매장을 언급하는지 매주 확인하고, AI 유입 트래픽도 측정합니다.
- **경쟁사 비교** — 동일한 GEO 신호를 경쟁사 상품 페이지와 나란히 비교합니다.

모든 기능에 대한 전체 안내는 [시작하기](getting-started.html)를, 정확한 플랜 한도는 [FAQ](faq.html)를 참고하세요.
